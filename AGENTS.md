# Agent Instructions

## Preferences

- Respond to users in their message language; use English for all other outputs unless explicitly instructed otherwise.

## Skills

- Always place newly created skills in the `skills/` folder of the current project.
- When creating a skill, also add an entry for it in `README.md` and create a corresponding documentation file under the `docs/` folder.

## Documentation

- Documents in the `docs/` folder are written for human readers: use a friendly, natural tone.
- Documents in the `skills/` folder are written for agents: use concise, precise, imperative, and formal language.
